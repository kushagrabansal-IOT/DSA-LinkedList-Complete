## рҹҺҜ Learning Objectives
- Build Singly, Doubly, Circular Linked Lists from scratch
- Master Floyd's Cycle Detection (phase 1 + phase 2)
- Implement LRU Cache with O(1) get/put
- Solve reverse, merge, partition problems in O(1) space
- Apply dummy node and two-pointer patterns

## рҹ“Ӣ Problem Statements
### P1 вҖ” Reverse: `1вҶ’2вҶ’3вҶ’4вҶ’5` becomes `5вҶ’4вҶ’3вҶ’2вҶ’1`
### P2 вҖ” Floyd Cycle: Detect if cycle exists + find entry point
### P3 вҖ” Merge Sorted: `[1,3,5]+[2,4,6]=[1,2,3,4,5,6]`
### P4 вҖ” Find Middle: `[1,2,3,4,5]вҶ’3` (one pass)
### P5 вҖ” Remove Nth End: Remove 2nd from end of `[1,2,3,4,5]вҶ’[1,2,3,5]`
### P6 вҖ” K-Group Reverse: k=2, `[1,2,3,4,5]вҶ’[2,1,4,3,5]`
### P7 вҖ” LRU Cache: O(1) get/put with eviction
### P8 вҖ” Palindrome LL: `[1,2,3,2,1]вҶ’True`
